# Movie Data Engineering Pipeline

This project implements an end-to-end data engineering pipeline using the MovieLens dataset.

## Technologies Used
- AWS S3
- Python
- Pandas
- PostgreSQL

## Pipeline Steps
1. Upload raw dataset to AWS S3
2. Extract data from S3
3. Transform and clean dataset
4. Load processed data into PostgreSQL
5. Run entire pipeline using pipeline.py
